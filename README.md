# FoodHub Analysis

Exploratory data analysis project for understanding customer orders, restaurant performance, cuisine patterns, delivery behavior, and actionable business insights.

## Structure

- `notebooks/` - analysis notebooks
- `data/` - local datasets, not committed unless explicitly allowed
- `src/` - reusable analysis helpers
- `reports/` - written findings and summaries
- `figures/` - generated charts and images

## Goals

- Perform clean exploratory data analysis
- Practice data storytelling
- Produce a clear, reproducible project suitable for GitHub
